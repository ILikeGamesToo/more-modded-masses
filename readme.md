More Modded Masses aims to provide a bunch of default masses to blocks added by mods, it started from a GitHub PR that lead me to [Mogwumps vs\_masses](https://github.com/Mogwump/Valkyrien-Skies-2/tree/1.20.1/main/common/src/main/resources/data/valkyrienskies/vs_mass/compat), which I tweaked/added to, and turned it into a datapack.

If something is broken or has an incorrect weight, fix it.
If you would like a mod to be added, add it.
